# Data

Этот каталог содержит скрипты для загрузки и предварительной обработки данных.

## Скрипты

- **load_data.py**: Скрипт для загрузки данных из CSV-файлов.
- **preprocess_data.py**: Скрипт для предварительной обработки данных, включая обработку пропусков и масштабирование.

## Как использовать

1. **Загрузка данных**:
   - Откройте файл `load_data.py` и укажите путь к вашему файлу CSV.
   - Запустите скрипт командой: `python load_data.py`.

2. **Предварительная обработка данных**:
   - Убедитесь, что данные загружены.
   - Запустите скрипт командой: `python preprocess_data.py`.
